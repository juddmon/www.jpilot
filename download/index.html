J-Pilot

Desktop Organizer Software for the Palm Pilot

Downloads

The latest J-Pilot source code is on github, or click in the GitHub icon at the bottom of the page. The new GTK3 release is on the feature-gtk3 branch

There is a PPA for Debian, Ubuntu 20.04 and 22.04.

To install run curl -s https://packagecloud.io/install/repositories/judd/jpilot/script.deb.sh | sudo bash
