J-Pilot

Desktop Organizer Software for the Palm Pilot

Latest News

J-Pilot is a desktop organizer application for PalmOS devices. It is meant to be an alternative to the Palm Desktop for those who run the most popular Operating Systems in the World, Linux and Unix.  Note, that most common does not equate to most popular.
